gitの練習をしていきます。
基本的なワークフローを学習します。
・git add / ステージに乗せる
・git commit / コミットする
・git rm / 削除する
・git status / 現在の状態を確認する
・git diff / 差分を確認する
・git log / 変更を確認する
・git push / プッシュする
・gitignore / 管理しないファイルを外す
・git rm [file]/ ローカルファイルを削除する
・git rm --cached [file]/ gitのファイルのみ削除する
・git reset [file] / gitの管理を戻す
・git checkout [file] / gitの管理もローカルファイルも戻す
